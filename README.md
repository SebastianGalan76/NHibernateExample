# NHibernateExample

Projekt stworzony w języku C# z wykorzystaniem NHibernate oraz Windows Form. Projektem jest podstawowy system do zarządzania biblioteką, który umożliwia m.in.: <br>
- Dodawanie i usuwanie klientów. <br>
- Edytowanie danych klienta. <br>
- Wyszukiwanie klientów po imieniu/nazwisku/adresie e-mail. <br>
- Dodawanie, usuwanie i wyszukiwanie książek. <br>
- Możliwość wypożyczania oraz zwracania książek przez klientów. <br>

Materiał pokazujący działanie programu: https://www.youtube.com/watch?v=g2lcTT0ZLp4
